Este projeto é um **site institucional / portfólio para uma fotógrafa especializada em casais e casamentos**, desenvolvido com foco em **apresentação visual do trabalho, experiência do usuário e captação de clientes**.

O objetivo do site é apresentar o estilo da fotógrafa, exibir o portfólio e facilitar o contato com novos clientes interessados em ensaios ou cobertura de casamento.

---

#### [◉°] Funcionalidades

O site possui uma estrutura completa de **portfólio profissional para serviços de fotografia**:

- **Hero de apresentação**  
  Apresenta a fotógrafa logo na seção inicial, destacando sua identidade visual e proposta artística.

- **Navegação por âncoras**  
  Header fixo que permite navegar rapidamente entre as seções do site.

- **Portfólio visual**  
  Galeria com diversas fotografias de ensaios e casamentos.

- **Seção Sobre**  
  Área dedicada a apresentar a fotógrafa, sua história e abordagem profissional.

- **Experiência / Processo**  
  Explica como funciona o atendimento, desde o primeiro contato até a entrega das fotos.

- **Pacotes de serviços**  
  Exibição clara dos investimentos disponíveis, separados entre:
  - Ensaios de casal
  - Cobertura de casamento

- **Formulário de contato**  
  Permite que o visitante envie uma solicitação preenchendo:
  - Nome
  - E-mail
  - Telefone
  - Data do evento
  - Pacote desejado
  - Mensagem

- **Feedback visual ao enviar**  
  Exibição de **toast de sucesso** após o envio do formulário.

- **Redes sociais no footer**  
  Links diretos para **Instagram, Pinterest e contato**.

- **Design responsivo**  
  Interface adaptada para **desktop e mobile**, incluindo **menu mobile no header**.

---

#### [◉°] Objetivo do Projeto

Este projeto foi desenvolvido para praticar e demonstrar habilidades em:

- Construção de **landing pages modernas**
- **Componentização com React**
- Uso de **TypeScript no front-end**
- Criação de layouts responsivos com **Tailwind CSS**
- Estruturação de um **site institucional voltado para serviços**
